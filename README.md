# 💡 База знаний по фронтенду / Frontend Knowledge Base

Добро пожаловать в структурированное хранилище материалов по веб-разработке.  
Welcome to a structured repository of web development materials.

---

## 📖 О проекте / About This Project

Этот проект создан на основе **Vue Книжный Клуб** — сообщества, где мы изучаем и обсуждаем лучшие книги, статьи и практики веб-разработки.  
This project is built on the foundation of the **Vue Book Club** — a community where we study and discuss the best books, articles, and practices in web development.

База знаний будет постоянно **пополняться** новыми материалами, конспектами, ссылками и примерами кода по мере изучения новых тем.  
The knowledge base will be continuously **updated** with new materials, summaries, links, and code examples as we explore new topics.

## 🚀 Как пользоваться / How to Use

1. Переходите в нужную папку через ссылки выше. / Navigate to the desired folder using the links above.
2. В каждой папке есть свой `README.md` с описанием содержимого. / Each folder contains its own `README.md` with a description of its contents.
3. Ссылки на внешние ресурсы находятся в файлах `links.md`. / Links to external resources are in the `links.md` files.
